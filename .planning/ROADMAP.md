# Roadmap: Badminton Court Booking System

## Milestone 1: Foundation & Core Flow

### Phase 1: Interactive Design & UI Foundation
- Design System: Dark theme, Thai typography (UTF-8).
- Landing Page: OTP Login UI.
- Member Onboarding: Profile form.

### Phase 2: Booking Experience
- Court Map Visualization (Badminton/Tennis).
- Date & Time Selector (14-day limit, hourly slots).
- Real-time availability simulation/API.

### Phase 3: Checkout & Payments
- Checkout page UI.
- Payment Timer (15 min).
- Mock Payment Integrations (QR, Wallet, CC).

## Milestone 2: Admin & Management

### Phase 4: Admin Dashboard
- Allotment Management (Open/Close slots).
- Pricing Management.
- Admin Calendar View.
- Transaction Reports.

### Phase 5: Verification & Production Polish
- End-to-end testing of the booking flow.
- Performance optimization.
- Deployment readiness.

---
*Roadmap is subject to evolution as implementation progresses.*
